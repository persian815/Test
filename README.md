# MLR Review Tool

Medical Affairs 검토 요청 전, 외부 홍보물의 주요 MLR 리스크를 Rule-based로 사전 점검하는 Next.js 프로젝트입니다.

## 실행

```bash
npm install
npm run dev
```

## Vercel 배포

```bash
npm run build
vercel
```

## 지원 파일

- 홍보물: PDF, PPTX, DOCX, TXT, MD
- Reference: PDF, DOCX, TXT, MD
- 파일당 200MB UI 제한

## 주의

본 결과는 Rule-based 예비 검토입니다. 최종 검토 및 승인 권한은 Medical Affairs에 있습니다.
